Syntax

  • Here in the first line React package is imported from the library and the component which is the main class from the react package
  • In the second line the components are imported from the react native package.The components imported for this file are Button,View and Text
  • In the third line a child class App is extended from the base class Component
    • Note:All the classes used in React.js are extended from a base class
  • Render functions same as display function in javascript.Values are always returned in the render function using return function
  • All the styling and adding the componets is done in return function.All the styling is in the form of HTML tags
  • The Button tag is used to create a button and its attribute "title" is used to give caption inside the button
  • View tag is used to give styling to the button because Button tag does not have its own styling attributes
  • The "style" attribute of the View tag is used to give position to the button
